# TV Tracker CLI

TV Tracker is a CLI that tracks the shows you watch episode by episode via the command line

TV Tracker lets you:
  • rate shows
  • maintain a queue
  • finish episodes
  • interact with show data

## Installation
Simply fork and clone this repository to begin using TV Tracker in the command line.
